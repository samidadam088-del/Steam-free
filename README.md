# 🎮 Steam Free - منصة حسابات Steam المجانية

منصة عصرية وآمنة لمشاركة حسابات Steam مجانية مع واجهة مستخدم احترافية.

## ✨ المميزات

- ✅ **نظام مصادقة آمن** - تسجيل دخول وتسجيل جديد
- ✅ **إدارة الحسابات** - إضافة وتعديل وحذف الحسابات
- ✅ **نظام التقييمات** - تقييم وتعليقات على الحسابات
- ✅ **بحث متقدم** - البحث عن الحسابات والألعاب
- ✅ **واجهة عصرية** - تصميم احترافي وسريع
- ✅ **دعم عربي كامل** - واجهة بالعربية فقط
- ✅ **قابل للتوسع** - هيكل نظيف وقابل للتطوير

## 🚀 البدء السريع

### المتطلبات
- Node.js 14 أو أعلى
- MongoDB محلي أو سحابي
- npm أو yarn

### التثبيت

\`\`\`bash
# استنساخ المشروع
git clone https://github.com/samidadam088-del/Steam-free.git
cd Steam-free

# تثبيت المكتبات الخلفية
npm install

# تثبيت مكتبات الواجهة الأمامية
cd frontend
npm install
cd ..
\`\`\`

### إعداد البيئة

\`\`\`bash
# نسخ ملف الإعدادات
cp .env.example .env

# تعديل .env بإضافة بيانات قاعدة البيانات والمتغيرات
\`\`\`

### التشغيل

\`\`\`bash
# تشغيل الخادم والواجهة معاً
npm run dev

# أو
npm start  # الخادم
npm run client  # الواجهة (في نافذة منفصلة)
\`\`\`

الموقع سيكون متاحاً على: \`http://localhost:3000\`
الخادم سيكون متاحاً على: \`http://localhost:5000\`

## 📚 API المتاحة

### المصادقة
- \`POST /api/auth/register\` - تسجيل جديد
- \`POST /api/auth/login\` - تسجيل دخول
- \`GET /api/auth/me\` - الحصول على بيانات المستخدم (محمي)

### الحسابات
- \`GET /api/accounts\` - الحصول على جميع الحسابات
- \`GET /api/accounts/:id\` - الحصول على حساب معين
- \`POST /api/accounts\` - إضافة حساب جديد (محمي)
- \`PUT /api/accounts/:id\` - تعديل حساب (محمي)
- \`DELETE /api/accounts/:id\` - حذف حساب (محمي)
- \`GET /api/accounts/search/:keyword\` - البحث عن حسابات

### التعليقات
- \`GET /api/comments/account/:accountId\` - الحصول على تعليقات الحساب
- \`POST /api/comments\` - إضافة تعليق (محمي)
- \`DELETE /api/comments/:id\` - حذف تعليق (محمي)
